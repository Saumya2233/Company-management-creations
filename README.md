
# Company Portal Application

##  Overview
This project is a full-stack application built using:
- Backend: NestJS, PostgreSQL, TypeORM
- Frontend: Next.js, React Query, React Hook Form, Yup
- Authentication: JWT with session management

##  Features
- Phone number login with OTP (123456)
- JWT-based authentication
- Session management
- Company listing
- Create company functionality
- Logout functionality
